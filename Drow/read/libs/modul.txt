 checkf; -  до ; або \n - проверить есть ли файл по адресу А.
 cutf; - переместить файл с дареса А в адрес Б.
 delf; - удалить файл по адресу А.
 clearf; - очистить файл по адресу А.
 open; - открыть файл внешно, по адресу А.
 start; - чекнуть файл А и открыть файл А если он есть, иначе чекнуть файл Б и открыть если он есть. - сказать какой файл открыл.
